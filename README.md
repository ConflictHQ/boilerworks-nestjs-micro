# Boilerworks NestJS Micro

> NestJS 11 API-only TypeScript service. No frontend.

**Status:** Planned

NestJS 11 in API-only mode for teams that want TypeScript end-to-end with a structured, decorator-based framework. Built-in dependency injection, guards, interceptors, and pipes give you enterprise patterns without the enterprise bloat. Ideal for microservices, BFF layers, and TypeScript-first API backends.
